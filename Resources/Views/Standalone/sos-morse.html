Sos & Morse

An all-in-one multipurpose utility app that you can use for education too. Woo!

THE S.O.S BUTTON

It simply starts an S.O.S signal with the iPhone's flash. If after 15 seconds the signal is not stopped, an e-mail is auto-composed with the current location and time since the signal was started.
See below for further details.

LANDSCAPE

Pressing this button presents a text box where you can enter any text you want and it will be turned into morse code with the iPhone's flash.
Neat educative perk, ain't it?
The slider is there if you want a simple strobe.

SAVE THREE EMAILS

Here you can save up to three e-mail addresses so the app can auto-complete them when composing the message.
The message is auto-composed, all you have to do is press Send. In case of an emergency, it might prove useful.
